INFORMATION on WebDAV

 

Tim Berners-Lee developed WWW, the web browser can both read and write web pages. But as WWW gain popularity, later development limited the web browser to a read-only viewer of web content.

WebDAV came to be the extension of HTTP when Jim Whitehead lead the Internet Engineering Task Force of W3C in 1996, to discuss the problem of distributed authoring of the World Wide Web. The WebDAV working group focused to standardize extension for distributed authoring. A different working group focus effort on versioning. WebDAV finalized the extension protocol in 2007 after the Internet Engineering Steering Group accepted RFC 2518.

More information on WebDAV specifications and related protocol can be found in WebDAV Resources site. In addition, Jim Whitehead also provided information on IETF WEBDAV Working Group site.
